project nest

Summary

Project Nest was a proprietary codebase, ASP.NET MVC website. I worked on converting about 20 pages into a responsive layout and grid system to make it mobile friendly and improved the speed of the website by over 20%. The redesign allowed for visitors to access their account details from any device, while simplifying the user flow process and increase the login rate. Responsible for planning, prototyping UX/UI designs, front-end coding, and browser support testing. Code written to support IE9+. Demo and wireframes available upon request.
