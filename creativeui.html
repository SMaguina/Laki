Inspiration

Being a video junkie, I've always wanted to learn how to make those sleek, responsive video header displays that you see on websites like Airbnb and GoPro. It was a challenge figuring out the code from scratch that would return a width/height that fits the screen appropriatley in Bootstrap. The source code is on my Codepen here.

Video Header Animation

The video animation was customized and rendered to be compatible across all major browsers with .ogv and .webm files. On mobile, the video defaults to a static photo (intentionally). Bootstrap framework for HTML is used, no Javascript needed.

Tools:

Adobe After Effects, Photoshop, Firefogg,
HandBrake, Bootstrap, HTML/CSS
